# Record Shop Backend

Welcome to the Record Shop Backend! This project is designed to manage the inventory and sales of a record shop. 
It provides a RESTful API for handling various operations such as adding new records, updating existing records, and processing sales.

#### Features:
- List albums in stock
- Get album by Id
- Add new albums to the database
- Update album details
- Delete albums

##### Nice to have features:
- List albums by artist
- List albums by release year
- Get album info by album name

#### In-Memory Database:
- Store album information in an in-memory database.
- This will allow data to be reset on startup, making it easier to test and develop without persistent storage.
